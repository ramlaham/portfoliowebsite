Portfolio

Most Recent Work

Interactive Chessboard (ICB)

In the process of manufacturing and leading the development of a real-time embedded smart chessboard with a team of 3 other engineers. This system integrates a 64-cell sensor matrix with LED feedback. Integrated an embedded AI chess engine for competitive play and coaching. Seamlessly merging the physical chessboard with our online 3D platform enabling real-time multiplayer, progression tracking, and a comprehensive analytics dashboard. Achieved 99% piece detection accuracy through sensor calibration and signal processing. Migrated the microcontroller from Arduino to ESP32 to support design-for-manufacturability (DFM). Technologies: C / C++, Python, ESP32, Three.js, Node.js, Firebase, Embedded Systems and Real-Time Processing.

Biometric Electoral System

Designed a secure biometric authentication system for electoral voting. Worked on hardware integration of the fingerprint sensor on Raspberry Pi; from device bring-up to validation, and end-to-end testing. Implemented AES encryption pipeline to protect biometric and voter data during processing and at rest. Developed the threat model (STRIDE) to identify and mitigate security risks. Built a responsive web application using TypeScript, React, and Vite.js to display election options and real-time results. Demonstrated expertise in secure system design, identity verification, and sensitive data protection. Technologies: Raspberry Pi, Fingerprint Scanner, AES Encryption, Python, C++, TypeScript, React, Vite.js, Secure System Design and Threat Modeling.

Access-19

Accessible Test-Strip Identification Mobile App. Developed an iOS / Android application with a team of 3 other engineers to assist visually impaired users in interpreting test-strip results using machine learning. Integrated a TensorFlow ML model for image-based test classification (Positive / Negative / Inconclusive results). Implemented audio output and intuitive React Native UI / UX with animations. Gathered requirements through iterative client meetings. Presented at University Engineering Design Day. Technologies: React Native, JavaScript, TensorFlow, Machine Learning, Computer Vision, Mobile Development, Accessibility Design.

PC Builders

Developed a Java & Kotlin-based Android application for managing PC hardware inventory & system configuration with a team of 4 other engineers. Conducted regular client meetings to gather requirements and incorporate feedback into the development cycle, meticulously tailoring it to client specifications. Implemented inventory management features including stock visualization, real-time updates, and CRUD operations. Integrated with Google Firebase for efficient database management and data persistence. Built a responsive UI using Android Jetpack libraries. Led version control and CI / CD workflows using GitHub. Technologies: Java, Kotlin, Android Jetpack, Google Firebase, and Git.
